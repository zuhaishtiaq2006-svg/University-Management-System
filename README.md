# University-Management-System
A desktop-based application built with Java Swing and MySQL to streamline academic and administrative operations of a university. This project integrates multiple modules for managing students, teachers, examinations, and fee structures with a modern purple-themed UI.

✨ Features
Student Management: Add, update, and view student details.
Teacher Management: Maintain faculty records with ease.
Examination Module: Enter marks, view results, and manage examination details.
Fee Structure & Payment: Display fee structures and allow students to pay fees with proper record-keeping.
About Section: Show project details and team information in a professional layout.

🎨 UI Highlights
Modern purple-themed interface with consistent design across modules.
Compact, well-aligned forms for better usability.
Auto-resizing tables with light purple scrollbars for a premium look.
Hover effects on buttons for interactive feedback.

Language: Java
GUI Framework: Swing
Database: MySQL
Libraries: DbUtils for table handling
